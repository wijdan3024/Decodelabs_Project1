# Personal Portfolio Website

## Project Overview

This project is a personal portfolio website developed using HTML and CSS. The purpose of this website is to present my professional profile, technical skills, academic background, projects, and contact information in a visually appealing and user-friendly manner.

The website is designed with a modern dark theme and follows responsive web design principles to ensure compatibility across different screen sizes and devices.

## Features

- Modern and professional user interface
- Responsive design for desktop, tablet, and mobile devices
- Navigation bar for easy access to different sections
- Hero section with profile image and introduction
- About Me section
- Skills section
- Education section
- Projects section
- Achievements section
- Contact section with clickable LinkedIn and GitHub links
- Attractive hover effects and smooth user experience

## Technologies Used

- HTML5
- CSS3

## How to Run the Project

1. Download or clone the project files.
2. Open the project folder in Visual Studio Code.
3. Open the `index.html` file in any web browser.

Alternatively:

- Install the Live Server extension in Visual Studio Code.
- Right-click on `index.html`.
- Select **Open with Live Server**.
